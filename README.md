## Image Renamer Instructions

### Install Prerequisites

1. Download exiftool (as a zip file) from https://exiftool.org/
2. Unzip it.
3. Rename the 'exiftool(-k).exe' file to exactly 'exiftool.exe'.
4. Move the 'exiftool.exe' file and the 'exiftool_files' folder into the same folder as these scripts.

### First Time Setup

Run the "image_renamer_launcher.bat" file by double clicking it.
This will create the necessary folders to be used by the program.

### How to Use

1. Copy the images you want to rename into the "Pending" folder.
2. Run the "image_renamer_launcher.bat" file by double clicking it.

The program will rename the images in the "Renamed" folder.
The original images will be copied to the "Original" folder.
The "Pending" folder will be emptied as it processes the images.

To process more images, copy them into the "Pending" folder and run the "image_renamer_launcher.bat" file again.
